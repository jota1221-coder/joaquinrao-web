# Joaquín Rao — Landing personal

Sitio personal y carta de presentación. Analista de datos y desarrollo para PyMEs en Zona Norte, Buenos Aires.

Una sola página autocontenida (`index.html`): sin build ni dependencias. Campo de estrellas y cajas de detección en `<canvas>`, animaciones con `IntersectionObserver`, respeta `prefers-reduced-motion`. Tipografías vía Google Fonts (Fraunces, Work Sans, JetBrains Mono).

## Ver en local

Abrí `index.html` en el navegador, o levantá un server estático:

```
python -m http.server 8777
```

y entrá a http://127.0.0.1:8777
